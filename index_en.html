Pointer Meter Automatic Detection and Identification

First, collect the meter picture data from the gas station, sort and mark it. Then, use the single-stage target detection algorithm CornerNet Lite to train and detect the meter. Next, use the end-to-end OCR algorithm E2E to train and identify the numbers and units in the meter, the pointer is train and segmented by the semantic segmentation algorithm PSP. The intermediate results obtained by deep learning are used in combination with other algorithms to calculate the final number of the meter.The architecture of entire algorithms are deployed on ROS, The data acquired from the camera, and calculated in patrol robot, finally the results are published to ROS, which is then sent to the client.

Switch Detection of Power Station

Firstly, switch and contact data of the power station are collected, label and expand the data, use the single-stage target detection algorithm YoloV3 to train and detect the switch and contacts of the power station, and use Node to publish the results to the ROS system.

Pedestrian Tracking Based on Spencer

Use RGBD camera to follow the people, first use YoloV3 to detect the people, then use DeepSort to extract the features of the people, and by using Kalman filtering to estimate the position of the next frame and assign the ReID, and use the Hungarian algorithm to match this frame and next frame of detection features when tracking. Finally publish the position information of the match result of each frame to the ROS system.

driveway line edge detection

First install a RealSense D415 camera on the patrol robot to collect data, organize and label the collected data, then use DeepLabV3Plus to train and semantically segment the driveway. Edge detection was carried out on the driveway area after segmentation, to extract the edge points and the corresponding depth points are saved to Node. Finally the results are published to the ROS system using Node.

Goods Detection in Tray of Airport

Intel RealSense D415 is adopted as the detection device. Firstly, the depth information of D415 is used to lock the edge area of the tray, and then the area inside the tray is segmented and calculate the patch information of this area. Finally, according to the difference of local variance and combined with the depth information inside the tray to detect whether the goods in the tray.

OCR Based on PDF File

The entire algorithm architecture is divided into five parts. The first part is PDF convert to image. The second part is text detection, firstly, CTPN is used to detect line text, and then combine other algorithms to segment each text, and count the position information of each text; The third part is the character recognition, the backbone of the recognition network is dpn92, and the training characters are from GB2312 encoding; The fourth part is text generate to txt file, which is written into a txt file according to the recognition result and position information of each word; The fifth part it is txt file convert to Word file, that is the content in the txt file is written into the Word file. The entire algorithm architecture is deployed on the cloud server, the front end is deployed with Flask. The server response is obtained by sending a PDF document request. the server uses multi-threaded parallel recognition to return the results to the front end. When testing a 400-word pure PDF document, the accuracy rate can reach about 80%.

Face Recognition Development Based on Fluorite Camera

We using the fluorite camera to go on secondary development, At the beginning, the face recognition module of dlib library was used, and later, FaceNet was used for face recognition. In detail, firstly, MTCNN is used to detect the face, then find the key points of the face, align the face, calculate the 512-dimensional face feature vector, and finally compare the feature vector with the feature vector in the face library, finally, return the comparison result to the server. Without considering the interference of occlusion and bright light, the correct recognition rate of the camera can reach about 90%.

Face Expression Recognition

First use Yolo to detect the face, then use 68 descriptors to locate the facial features and set them as key points, combine these key points for classification and discrimination, and finally recognize the facial expression through the classification and discrimination results. There are four types of expressions to be recognized: angry, surprised, happy, and normal. The accuracy of expression recognition can reach more than 90%.

Enhancement and Haze Removal in Traffic Surveillance Video

The goal is improve the imaging quality in traffic surveillance video by use image enhancement algorithm. Firstly, i developed a local histogram statistical algorithm, and then i developed a fuzzy domain edge contrast enhancement algorithm, the imaging enhancement effect is obvious. In the haze environment, a contrast-constrained adaptive histogram algorithm is used first, and then a new fast dehazing algorithm is developed based on the principle of Dark Channel and the atmospheric scattering model combined with PCA. The dehazing effect is obvious, and the algorithm can meet the real-time processing needs.
